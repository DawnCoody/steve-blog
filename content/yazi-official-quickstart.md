# Yazi 使用方式：按官网 Quick Start 一页看懂

这份内容主要根据 [Yazi 官方 Quick Start](https://yazi-rs.github.io/docs/quick-start/) 整理，并补充了 [Installation](https://yazi-rs.github.io/docs/installation/) 页面里和实际使用直接相关的依赖说明。

重点不是讲原理，而是讲这几件事：

- 启动后该怎么用
- `y` 应该怎么配置
- 哪些键最常用
- 多标签怎么切
- 哪些依赖会在 Yazi 里自动生效

## 先记住这 3 个点

- `yazi`：最基础的启动方式
- `y`：官网推荐的 shell wrapper 启动方式
- `q / Q`：退出时切换目录 / 不切换目录

## 最小启动方式

官网 Quick Start 的第一步就是：

```bash
yazi
```

进入后：

- 按 `q` 退出
- 按 `F1` 或 `~` 打开帮助

## 为什么更推荐用 `y`

官网建议给 shell 配一个 `y` wrapper。这样你退出 Yazi 后，终端会自动切到你最后停留的目录。

如果你是 `zsh`，把下面这段加入 `~/.zshrc`：

```bash
function y() {
  local tmp="$(mktemp -t "yazi-cwd.XXXXXX")" cwd
  command yazi "$@" --cwd-file="$tmp"
  IFS= read -r -d '' cwd < "$tmp"
  [ "$cwd" != "$PWD" ] && [ -d "$cwd" ] && builtin cd -- "$cwd"
  rm -f -- "$tmp"
}
```

然后执行：

```bash
source ~/.zshrc
```

以后就这样启动：

```bash
y
y ~/Downloads
```

退出时的区别：

- 按 `q`：退出后切换当前终端目录
- 按 `Q`：退出后不切换当前终端目录

## Yazi 的实际使用流程

如果你第一次上手，建议按这个顺序练：

### 1. 启动并进入目录

- 用 `y` 或 `yazi` 启动
- 如果已经知道目标目录，可以直接 `y ~/Downloads`

### 2. 先学会浏览目录

- `j / k`：上下移动
- `l`：进入目录
- `h`：回到父目录

这是最基础的一组键。

### 3. 学会最常见的文件操作

- `Space`：选中
- `y`：复制
- `x`：剪切
- `p`：粘贴

复制和移动是 Yazi 里最高频的一组操作。

### 4. 找东西时先分清是“找名字”还是“找内容”

- `s`：用 `fd` 按文件名搜索
- `S`：用 `ripgrep` 按文件内容搜索

### 5. 目录太多时，用跳转功能

- `z`：通过 `fzf` 快速定位当前目录树里的文件或目录
- `Z`：通过 `zoxide` 跳到历史常用目录

### 6. 需要并行处理目录时开标签页

- `t`：开新标签页
- `[` / `]`：切换标签页
- `1-9`：直接跳到对应标签页

## 官网快捷键整理

下面按场景重新整理了一遍，查起来会比直接翻官方原文更顺手。

### 导航

- `j / k`：向下 / 向上移动光标
- `h / l`：回到父目录 / 进入当前目录
- `g g`：跳到顶部
- `G`：跳到底部
- `J / K`：在预览区向下 / 向上快进 5 单位
- `z`：借助 `fzf` 跳目录或定位文件
- `Z`：借助 `zoxide` 跳历史目录
- `g Space`：通过交互输入跳目录或定位文件

### 选择与文件操作

- `Space`：选中或取消选中当前文件
- `v / V`：进入选择模式 / 取消式选择模式
- `Ctrl+a`：全选
- `Ctrl+r`：反选全部文件
- `Esc`：取消选择
- `Enter / o`：打开当前选中文件
- `O`：交互式打开文件
- `Shift+Enter`：交互式打开文件，部分终端可能不支持
- `y / x / p`：复制 / 剪切 / 粘贴
- `P`：强制覆盖粘贴
- `Y / X`：取消复制或剪切状态
- `d / D`：移到回收站 / 永久删除
- `a`：新建文件；如果输入名字以 `/` 结尾，则创建目录
- `r`：重命名
- `.`：显示或隐藏隐藏文件

补充几个很实用但容易忽略的键：

- `;`：运行 shell 命令
- `:`：运行 shell 命令并等待完成
- `- / _`：创建已复制文件的绝对路径 / 相对路径符号链接
- `Ctrl+-`：创建已复制文件的硬链接

### 查找与搜索

- `f`：过滤文件列表
- `/`：向下查找文件
- `?`：向上查找文件
- `n / N`：跳到下一个 / 上一个匹配项
- `s`：按文件名搜索，依赖 `fd`
- `S`：按文件内容搜索，依赖 `ripgrep`
- `Ctrl+s`：取消正在进行的搜索
- `Tab`：显示文件信息，辅助判断再操作

### 排序与复制路径

排序键：

- `, a` / `, A`：按字母正序 / 倒序
- `, n` / `, N`：按自然顺序正序 / 倒序
- `, m` / `, M`：按修改时间正序 / 倒序
- `, s` / `, S`：按大小正序 / 倒序
- `, e` / `, E`：按扩展名正序 / 倒序
- `, b` / `, B`：按创建时间正序 / 倒序
- `, r`：随机排序

复制路径键：

- `c c`：复制完整文件路径
- `c d`：复制目录路径
- `c f`：复制文件名
- `c n`：复制不带扩展名的文件名

### 多标签

官网把多标签看成“多个目录上下文”：

- `t`：以当前工作目录创建新标签页
- `1-9`：切到第 N 个标签页
- `[` / `]`：切换到上一个 / 下一个标签页
- `{` / `}`：与前一个 / 后一个标签页交换位置
- `Ctrl+c`：关闭当前标签页

什么时候最适合用多标签：

- 一个标签放项目根目录
- 一个标签放 `src`
- 一个标签放下载目录

这样切换会比来回进出目录更快。

## 这些依赖在 Yazi 里如何生效

这部分按官方 Installation 页面整理，只讲和实际使用直接相关的效果。

### 必需项

- `file`：Yazi 用它识别文件类型；没有它，很多预览和判断都会不稳定

### 推荐项

- Nerd Font：让图标正常显示。它不是命令，而是终端字体支持

### 会直接变成按键能力的依赖

- `fd`：给 `s` 用，按名字搜索文件
- `ripgrep`：给 `S` 用，按内容搜索文件
- `fzf`：给 `z` 用，快速定位目录或文件
- `zoxide`：给 `Z` 用，跳历史目录；官网注明它依赖 `fzf`

### 主要增强预览的依赖

- `ffmpeg`：视频缩略图
- `7-Zip`：压缩包提取与预览
- `jq`：JSON 预览
- `poppler`：PDF 预览
- `resvg`：SVG 预览
- `ImageMagick`：字体、HEIC、JPEG XL 预览

## 最适合新手记住的 12 个键

如果你不想一开始就记完整套，先把这组练熟，已经足够应付多数日常场景：

- `h j k l`：浏览目录
- `Space`：选中
- `Enter`：打开
- `y`：复制
- `x`：剪切
- `p`：粘贴
- `r`：重命名
- `a`：新建文件或目录
- `s`：按名字搜索
- `S`：按内容搜索
- `t`：开新标签页
- `q`：退出

## 两个一定要分清的点

### `q` 和 `Q` 的行为不同

如果你用的是官网推荐的 `y` wrapper：

- `q`：退出后会把终端切到你最后所在的目录
- `Q`：退出后不会切换目录

### `D` 是永久删除

官网把 `d` 和 `D` 分得很清楚：

- `d`：送进回收站
- `D`：直接永久删除

这个键一定要分清。

## 一个很实用的练习路径

如果你想真正用起来，而不是看完就忘，可以按这个顺序练 10 分钟：

### A. 打开下载目录

```bash
y ~/Downloads
```

练 `h j k l`。

### B. 选中两个文件后复制到新目录

- 用 `Space` 选中
- 按 `y` 复制
- 按 `a` 新建目录
- 进入新目录后按 `p` 粘贴

### C. 试一次名字搜索和内容搜索

- 按 `s` 搜文件名
- 按 `S` 搜文件内容

感受两者区别。

### D. 开一个标签页

- 按 `t` 新建标签页
- 用 `[` 和 `]` 切回去
